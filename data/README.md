# Data folder

Place your input GeoPackage (`.gpkg`) here with the name `df_urban_system_typology.gpkg`. It must contain the 13 predictor columns listed in the main README, a column `class` (target), and a geometry column.

If you do not have the data, you can request it from the author (abrilmargonari@gmail.com).

Do not upload large files to GitHub; keep them local.
